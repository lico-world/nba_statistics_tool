# <span style="color:#FFBA86">NBA Statistics Tool</span>
<span style="color:#C23373">**Abstract**:</span> Scraping, analysis and visualisation of NBA statistics from baskeball-reference or the official NBA site.

---

Libraries used:
 - <span style="color:#F6635C">Pandas:</span> Used to manage Dataframe (way to organize data received by APIs)
 - <span style="color:#F6635C">NBA API:</span> Used to send requests to the NBA API
 - <span style="color:#F6635C">Seaborn:</span> Used to plot good-looking graphs for a good data visualisation
 - <span style="color:#F6635C">Scipy:</span> Used to interpolate data (for good filling between curves for example)
 - <span style="color:#F6635C">Icecream:</span> Used for clean debug and log system
